## 인증 방법
1) 새로 PR을 생성하면 자동으로 템플릿이 생성됩니다. 
2) 매일 해당 문제를 풀고, PR을 올립니다. (https://github.com/Hello-There-JS-Study/Programmers/pull/17)
3) pr 템플릿에 이제까지 푼 문제를 체크표시 합니다.

4) pr 아래에 댓글로 `오늘 기억하고자 하는 메모` 를 작성합니다.

(방장이 매일 PR를 main에 머지합니다.) -> 이 부분은 다음 기수가서는 자동화할 계획입니다...

- 정리
  - issue: 문제 리스트
  - pr : 각자 문제 푼 거 올리는 곳


## 스터디 규칙
- only 온라인

| 일정 | 할일 | 
|-------|-------|
| 월 ~ 금   | 하루 30분 코딩테스트 풀이 -> 깃허브 공통 레포에 PUSH (제출 코드 + 기억해야될 메모 첨부) | 
| 토   | 해당 주 틀린 문제 2문제 복습 OR 새로운 1문제 |
| 일   | 휴식   |

- 2주 단위로 진행 (짫은 기간으로 집중도와 완주율 높이는 방식!)

## 폴더 구조, branck 및 commit message 규칙
- 폴더구조: 문제이름/이름.js ex) 더맵게/seonu.js
- pr
   - 제목: 날짜_문제번호_이름 ex) 260201_더맵게_seonu (제출 코드 내는 곳)
   - 내용: `기억하고자 하는 메모 내용` 적기 (메모 기록 남기는 곳)
   - 참고 링크: https://github.com/Hello-There-JS-Study/Programmers/pull/17
- branch 명: 본인 이름 ex) seonu
