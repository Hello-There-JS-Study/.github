# JavaScript Algorithm Study
JavaScript coding test study

- 기간: 2025.01.19 ~
- 목표: [프로그래머스 알고리즘 고득점](https://school.programmers.co.kr/learn/challenges?tab=algorithm_practice_kit) kit 전부 도전하는 스터디
- 언어: 자바스크립트

## 스터디 규칙
- only 온라인

| 일정 | 할일 | 
|-------|-------|
| 월 ~ 금   | 하루 30분 코딩테스트 풀이 -> 깃허브 공통 레포에 PUSH (제출 코드 + 기억해야될 메모 첨부) | 
| 토   | 해당 주 틀린 문제 2문제 복습 OR 새로운 1문제 |
| 일   | 휴식   |

- 2주 단위로 진행 (짫은 기간으로 집중도와 완주율 높이는 방식!)

## 폴더 구조, branck 및 commit message 규칙
- 폴더구조: 문제이름/이름.js ex) 더맵게/seonu.js
- pr
   - 제목: 날짜_문제번호_이름 ex) 260201_더맵게_seonu (제출 코드 내는 곳)
   - 내용: `기억하고자 하는 메모 내용` 적기 (메모 기록 남기는 곳)
   - 참고 링크: https://github.com/Hello-There-JS-Study/Programmers/pull/17
- branch 명: 본인 이름 ex) seonu


## 하루 30분 루틴
1) 문제 읽기 (0~5분)
    - 문제 천천히 읽기
    - 입력/출력 정리
    - 예제 직업 따라가 보기
  
2) 1차 구현 (5~15분)
    - 완벽 x
    - 일단 돌아가게 구현
    - console.log 적극 사용
    - 10분 넘으면 바로 해설 확인

3) 문법 정리
    - 오늘 사용한 JS 문법 정리
    - 예: Map, split, join, sort 등
    - 다음에 바로 써먹을 수 있게 기록
